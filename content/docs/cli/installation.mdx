## Prerequisites

Before you begin, ensure your environment meets the following requirements:

- **Standard Server Requirements**:
  - **RAM**: 32 GB
  - **CPU**: 2–4 cores
  - **Storage**: 50 GB

- **Software Dependencies**:
  - Node.js: `20.x`
  - Docker: `latest`
  - Redis: `latest`
  - Nomad: `1.6.3`
  - Consul Client: `https://cdn.brimble.io/consul.sh`

<Callout>
Ensure that all servers are accessible via SSH and have appropriate credentials configured.
</Callout>

---

## Installation Steps

### Step 1: Generate License Keys

1. Log in to your Brimble account.
2. Navigate to the **Licenses** section.
3. Generate a license key for your infrastructure.

> Keep your license key secure, as it is required for authentication during the setup process.

---

### Step 2: Provision Servers

Provision servers that meet the recommended specifications. Assign each server a public and private IP.

```js
{
  "host": "test-2",
  "username": "root",
  "key_path": "~/.ssh/id_rsa",
  "public_ip": "178.128.161.67",
  "private_ip": "10.106.0.4",
  "auth_method": "key_path",
  "region": "europe"
}
```



### Step 3: Provision Servers

Install the Brimble CLI globally using npm:
```package-install
npm install -g @brimble/cli
```

### Step 4: Configure the CLI
Create a configuration file (brimble.config.json) to define your server setup:
```json
{
  "servers": [
    {
      "host": "test-2",
      "username": "root",
      "key_path": "~/.ssh/id_rsa",
      "public_ip": "178.128.161.67",
      "private_ip": "10.106.0.4",
      "auth_method": "key_path"
    }
  ],
  "runner": {
    "port": 3000,
    "instances": 4
  },
  "monitoring": {
    "grafana_password": "password",
    "metrics_port": 9100
  }
}

### Step 5: Run the Setup Script
```js
brimble setup --config brimble.config.json --license <YOUR_LICENSE_KEY>
```

### Step 6: Verify Installation
Check the status of your setup using the CLI:
```js
brimble status
```
Example output:
```js
All servers configured successfully!
Nomad cluster initialized: ✓
Consul service running: ✓
Metrics setup completed: ✓
```
