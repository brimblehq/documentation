## Key Features Across All Plans

No matter which plan you choose, you’ll get access to **core features** that make Brimble the best choice for modern deployment workflows:

- **Git Deployment**: Connect your GitHub or GitLab repository and deploy in seconds.
- **Live Project Preview**: Share a live URL to preview your project instantly.
- **CLI Deployment**: Deploy from your terminal with the Brimble CLI.
- **Automatic HTTPS / SSL**: Secure your deployments with free SSL certificates.
- **Instant Deployment Rollbacks**: Revert to a previous version with one click.
- **External Integration Support**: Integrate with third-party tools like Slack and GitHub.
